Pasos a Seguir

Abre una terminal en tu computadora.

-Ejecuta el siguiente comando para instalar adb sudo apt-get install adb.

- Una ves instalado el programa adb en nuestro ordenador, abre una terminal y ejecuta el siguiente comando adb tcpip : 43232.

Cabe mencionar que los numeros despues de tcpip son el puerto al cual se conectar nuestro dispositivo por eso una ves despues de establecidos es recomentable que los anotes en algun lugar por que pronto seran utilizandos.
Tambien no es sumamente necesaio que establescas como puerto el numero 42232 tu elijes el puerto que quieras utlizar en mi caso utlizare ese, pero como ya mencione, puedes usar el que tu gustes o prefieras.

Posteriormente activaremos el depurador usb de nuestro dispositivo android, si no sabes como activar el depurador usb puedes visitar el siguiente enlace, para que aprendas como activarlo, activar depurador usb en android.

Despues de ejecutar el siguiente comando adb connect (nuemero-proxy:puerto).
Donde el proxy es el numero que nos ha proporcionado nuestro router, y el puerto es el numero que hemos asignado nosotros con el comando anterior adb tcpid:puerto. Un ejemplo como nos quedaria en una terminal seria el siguiente adb connect 192.168.1.1:43232 y preciona enter.
